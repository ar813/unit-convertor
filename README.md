# Unit Converter

This is my first class project in Python, assigned by my teachers from GIAIC. The project is a simple yet efficient unit converter built using Python and Streamlit, allowing users to convert between various units across multiple categories.

## Features

- Convert units across multiple categories:
  - Area
  - Energy
  - Frequency
  - Length
  - Mass
  - Data Transfer Rate
  - Digital Storage
  - Fuel Economy
  - Plane Angle
  - Pressure
  - Speed
  - Time
  - Volume
- Interactive UI using Streamlit
- Easy-to-use sidebar for selecting unit categories
- Instant conversion results

## Technologies Used

- **Python**
- **Streamlit**

## Live Demo

Try the application here: [Unit Converter](https://unit-convertor-python01.streamlit.app/)

## Usage

- Start the application using the link.
- Select a unit category from the sidebar.
- Choose the units you want to convert from and to.
- Enter the value you want to convert.
- Click on the "Convert" button to see the result.
